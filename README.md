# Exercise2

# How to run on Linux
```
git clone https://github.com/Sevelantis/Exercise2
cd Exercise/
mkdir bild
cd bild/
cmake ..
cmake --build .
```
# optional arguments: (double arg1, double arg2)
 - arg1 - Julian Date code
 - arg2 - Longitude ('-' means west)
```
./app 2451553.00069 98.7645
```
